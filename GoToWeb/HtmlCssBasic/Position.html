Позиционирование: - для наложение одного обьекта на другой - static - по умолчанию -при любом позиционировании, кроме static доступны дополнительные css свойства: top, bottom, left, right, z-index а также элементы выпадают из потока, но при каждом типе поз-ия это выпадение своё - когда заданы два противоположных свойства приоритет у top, left - relative - относительное позиционирование элемент выпадает из потока, но место элемента бронируется, и при этом элем. можно перемещать при помощи top, bottom, left, right например top: 20px - смещение на 20px сверху - absolute - абсолютное позиционирование, место за элементом не бронируется, 1)элементы ведут себя как inline-block (т.е. для блочныхэлементов - исчезает растягивание на всю ширину, для строчных элементов - появляется возможность задавать высоту и ширину) 2) при применении t,b,l,r - влияет как задано позиционирование родителя если у всех static - значит элемент позиционируется относительно body если у родителя(одного из) позиционирование например relative (отличное от static) - элемент позиционируется относ. родительского элемента 3)при прокрутке элемент смещается - fixed - фиксированное позиционирование похоже на absolute, с той разницей что позиционирование всегда отсчитуется от окна браузера(от viewbord) при прокрутке элемент не смещается - z-index - индекс расположение элемента по оси Z (когда произходите наложение элементов) чем больше индекс - тем ближе к поверхности 1) 0 - значение по умолчанию 2) -1 - прячет за родительский элемент - Растяжка: если у элемента не задана ширина, или высота - то задание left-right top-bottom растягивает элемент до соответствующих отступов - Центровка по горизонтали: postion:absolute & если задана ширина элемента, left:0, right:0, margin: 0 auto - Центровка по вертикали относ род. элемента: postion:absolute & если задана высота элемента, top:0, bottom:0, margin: auto 0 - если совместить : postion:absolute, width, height, left:0, right:0, top:0, bottom:0, margin: auto - overlay или paranja - блок перекрывающий всё изображение(что-то вроде popup)
